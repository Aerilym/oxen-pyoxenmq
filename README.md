# pylokimq

pybind layer for lokimq

## building

dependencies

- python3-dev
- cmake 
- pkg-config 
- liblokimq-dev


check the source out (recursive repo):

    $ git clone --recursive https://github.com/loki-project/loki-pylokimq
    $ cd loki-pylokimq

build:

    $ python3 setup.py build
    
install as user:

    $ python3 setup.py install --user
